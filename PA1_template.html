<li>Unzip the .zip file.</li>
<li>Read the .csv file defining the columns classes.
<ul>

</ol>

<h2>Questions in this Peer Assessment</h2>
<h3>What is mean total number of steps taken per day?</h3>
<p>We can see that our data consists of three variables:</p>
<ul>
<li><strong>steps</strong>: The number of steps for each interval.</li>
<li><strong>date</strong>: The day, month and year the data was taken.</li>
<li><strong>interval</strong>: The 5-minute interval of the day.</li>
</ul>
<p>For this part of the assignment, we can ignore the missing values (NA) in the dataset.</p>
<p>I make the aggregation of the steps by date using &#39;dplyr&#39; and a histogram of the total number of steps taken each day using &#39;ggplot&#39;. Plotted with a bin interval of 1000 steps.</p>
<pre><code class="r">day_total &lt;- tbl %&gt;%

</code></pre>
<pre><code class="r">ggplot(day_total, aes(x=total)) +

<h3>Imputing missing values</h3>
<p>As there are a number of days/intervals where there are missing values (coded as NA). The presence of missing days may introduce bias into some calculations or summaries of the data.</p>
<p>To verify if this bias is produced, first I calculate and report the total number of missing values in the dataset.</p>
<pre><code class="r">sum(is.na(tbl))
</code></pre>
<pre><code>## [1] 2304
</code></pre>
<p>There are <strong>2304</strong> NA&#39;s (8 days of the data).</p>
<p>Next, I fill in all of the missing values in the dataset using the mean for that 5-minute interval. I do this by binding a column with the means of steps per 5-minute interval and replacing every NA I find in an interval for its mean. Finally I subset the 3 original columns.</p>
<pre><code class="r">tbl_noNA &lt;- cbind(tbl, day_avg)
tbl_noNA$steps[is.na(tbl_noNA$steps)] &lt;- tbl_noNA$avg_steps[is.na(tbl_noNA$steps)]
tbl_noNA &lt;- tbl_noNA[, 1:3]
</code></pre>
<p>Next, I make a histogram of the total number of steps taken each day as we did with the original data with NA&#39;s.
I make the aggregation of the steps by date using &#39;dplyr&#39; and a histogram of the total number of steps taken each day using &#39;ggplot&#39;. Plotted with a bin interval of 1000 steps.</p>
<pre><code class="r">day_totalnoNA &lt;- tbl_noNA %&gt;%
group_by(date) %&gt;%
summarise(total = sum(steps, na.rm = T))
</code></pre>

<h3>Are there differences in activity patterns between weekdays and weekends?</h3>
<p>For this part, using the dataset with the filled-in missing values, I create a new factor variable in the dataset with two levels - &ldquo;weekday&rdquo; and &ldquo;weekend&rdquo; indicating whether a given date is a weekday or weekend day. I do this by:</p>
<ul>
<li>Create a new column using &#39;mutate&#39; with the abbreviated day of the week.</li>
<li>Use &#39;revalue&#39; to rename the factors to &#39;Weekday&#39; and &#39;Weekend&#39;.</li>
<li>Subset by each factor.</li>
<li>Aggregate the mean of the steps by interval of each set.</li>
<li>Rename columns.</li>
<li>Bind both sets.</li>
</ul>
<pre><code class="r">tbl_days &lt;- tbl_noNA %&gt;%
mutate(type_of_day = as.factor(format(date,&quot;%a&quot;)))
tbl_days$type_of_day &lt;- revalue(tbl_days$type_of_day, c(&quot;Mon&quot;=&quot;Weekday&quot;, &quot;Tue&quot;=&quot;Weekday&quot;, &quot;Wed&quot;=&quot;Weekday&quot;,
&quot;Thu&quot;=&quot;Weekday&quot;, &quot;Fri&quot;=&quot;Weekday&quot;, &quot;Sat&quot;=&quot;Weekend&quot;,
&quot;Sun&quot;=&quot;Weekend&quot;))
tbl_weekday &lt;- subset(tbl_days, type_of_day == &quot;Weekday&quot;)
tbl_weekend &lt;- subset(tbl_days, type_of_day == &quot;Weekend&quot;)
weekday_avg &lt;- aggregate(tbl_weekday$steps, list(tbl_weekday$interval,tbl_weekday$type_of_day), mean)
weekend_avg &lt;- aggregate(tbl_weekend$steps, list(tbl_weekend$interval,tbl_weekend$type_of_day), mean)
colnames(weekday_avg) &lt;- c(&quot;interval&quot;, &quot;type_of_day&quot;, &quot;avg_steps&quot;)
colnames(weekend_avg) &lt;- c(&quot;interval&quot;, &quot;type_of_day&quot;, &quot;avg_steps&quot;)
weekday_data &lt;- rbind(weekday_avg, weekend_avg)
</code></pre>
